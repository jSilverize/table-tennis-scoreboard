## Development
It's an Single Page Application maded with ReactJS.

### Environment
Recomended to use `yarn` instead `npm`, but you need thoose two globally installed in your environment
```
npm install -g yarn sass
```

### Dependencies
```
yarn
```

### Run locally
```
yarn dev
```

### Build
```
yarn build
```
